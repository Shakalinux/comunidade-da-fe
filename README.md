# Comunidade da Fé

O projeto **Comunidade da Fé** nasceu com o propósito de criar um espaço digital voltado à reflexão, ao estudo e ao crescimento espiritual.
A proposta é unir tecnologia e fé de maneira simples e acessível, oferecendo um ambiente onde as pessoas possam se conectar com a Palavra, estudar, compartilhar experiências e fortalecer sua caminhada cristã.

## Objetivo

Criar uma plataforma que una organização, clareza e propósito, proporcionando uma experiência acolhedora e fluida para o usuário.
O sistema oferece seções dedicadas a leituras bíblicas, estudos, versículos do dia e conteúdos que incentivam a vivência da fé no cotidiano.

## Conceito

Mais do que um projeto técnico, a Comunidade da Fé representa a união entre propósito e desenvolvimento.
Cada parte do sistema foi pensada para transmitir calma, leveza e significado, mostrando que a tecnologia também pode ser uma ferramenta para inspirar e aproximar as pessoas de algo maior.

## Tecnologias utilizadas

* Java
* Spring Boot
* Thymeleaf
* HTML, CSS e Bootstrap
* MariaDB
